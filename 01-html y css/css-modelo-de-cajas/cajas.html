Modelos de caja

Elementos de bloque

Los elementos de bloque, como los encabezados y los párrafos, ocupan todo el ancho disponible dentro de su contenedor y generan una nueva línea después de ellos.

Elementos en línea

Los elementos en línea, como <em> y <strong>, solo ocupan el espacio necesario para su contenido y se alinean horizontalmente con otros elementos en línea.
